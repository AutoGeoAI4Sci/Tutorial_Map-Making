# Contributing

Contributions are welcome if they improve practical learning about map-making, cartography, and reproducible geospatial visualization.

## Good Contributions

- lecture notes or summaries
- map design checklists
- reproducible examples in Python, R, QGIS, ArcGIS Pro, or web mapping tools
- before-and-after map redesigns
- links to open cartography references
- small datasets that are clearly licensed for teaching

## Attribution

Credit presenters, authors, data providers, and software projects clearly. If a resource comes from a lecture recording, mark whether your contribution is a summary, note, code example, or transcript.

## Sensitive or Restricted Materials

Do not upload private recordings, downloaded videos, unpublished slides, or restricted datasets unless the owner has explicitly approved public redistribution.
